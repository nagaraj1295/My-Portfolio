# Nagaraj C | Modern Portfolio

A high-performance, responsive portfolio website built with React 19, Tailwind CSS v4, and Framer Motion.

## 🚀 Features
- **Interactive UI**: Modern blue/white professional theme.
- **Advanced Animations**: Dynamic Hero typewriter effect and smooth scroll-reveal transitions.
- **Responsive & Future-Proof**: Innovative horizontal scroll for Skills, Certifications, and Projects on mobile viewports.
- **Consolidated Data**: Comprehensive professional profile integrated from multiple resume versions.
- **Social Integration**: One-click access to LinkedIn, GitHub, Instagram, Facebook, and YouTube.

## 🛠️ Tech Stack
- **Framework**: React 19
- **Styling**: Tailwind CSS v4 (CSS-first engine)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## 📦 Getting Started
1. **Clone the repository**: `git clone https://github.com/nagaraj1295/My-Portfolio.git`
2. **Install dependencies**: `npm install`
3. **Run development server**: `npm run dev`
4. **Build for production**: `npm run build`
